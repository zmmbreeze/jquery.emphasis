jQuery.emphasis.js

A javascript fallback for CSS3 "text-emphasis". It's require jQuery. It's support IE8+, Chrome 12+, Opera 10+, Firefox 4+ and Safari 5+.

Usage

Include jQuery and emphasis plugin into html.

<script src="jquery-1.10.2.min.js"></script>
<script src="jquery.emphasis.js"></script>
<script>
$('em').emphasis('dot');
</script>
